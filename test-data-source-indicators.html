🔍 Data Source Indicators Demo

Demo Purpose

This page demonstrates the visual indicators that show users whether their Additional Cost Breakdown data comes from:

  • ☁️ Cloud Data - Synced across all browsers
  • 💾 Local Data - Only available in current browser
  • 🔄 Mixed Data - Partial sync between cloud and local
  • 📭 No Data - No cost data available

💡 What Users See

☁️ Cloud Data (Supabase)
  • ✅ Data syncs across all browsers
  • ✅ Reliable and persistent
  • ✅ Shows item count from database
  • ✅ Green indicator with cloud icon
💾 Local Data (localStorage)
  • ⚠️ Only available in current browser
  • ⚠️ May be lost if browser data cleared
  • ⚠️ Shows "Sync to Cloud" button
  • ⚠️ Yellow indicator with hard drive icon
