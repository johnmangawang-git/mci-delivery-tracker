Individual DR Entries Test

This test verifies that each DR entry is preserved individually, even with the same DR number, because each has a unique Serial Number.

Test Scenario

Scenario: Multiple items with same DR number but different serial numbers

  • DR-001 with Serial Number SN-001
  • DR-001 with Serial Number SN-002
  • DR-001 with Serial Number SN-003
Expected Result

Should create 3 separate deliveries:

  • DR-001 (original)
  • DR-001-2 (second occurrence)
  • DR-001-3 (third occurrence)
